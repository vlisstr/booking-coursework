# Booking System Coursework

## Опис проєкту
Даний проєкт є курсовою роботою з дисципліни «Бази даних».
Метою роботи є розробка реляційної бази даних для системи бронювання ресурсів
з використанням PostgreSQL та Node.js.

---

## Мета роботи
Проєктування та реалізація бази даних, яка дозволяє зберігати інформацію
про користувачів, ресурси та їх бронювання.

---

## Використані технології
- PostgreSQL — система управління базами даних
- Docker — розгортання локального середовища
- pgAdmin — адміністрування бази даних
- Node.js (JavaScript) — взаємодія з базою даних
- SQL — створення та обробка даних

---

## Структура бази даних
База даних складається з наступних таблиць:
- **users** — користувачі системи
- **resource_types** — типи ресурсів
- **resources** — ресурси, доступні для бронювання
- **bookings** — інформація про бронювання

Між таблицями реалізовані зв’язки за допомогою зовнішніх ключів.
